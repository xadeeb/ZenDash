# 🚀 ZenDash | Personal Productivity Hub

**ZenDash** is a production-ready, minimalist dashboard built with a sleek Glassmorphism design and modular JavaScript architecture. It's designed to keep your focus sharp with essential productivity tools in one place.

---

## ✨ Key Features
* **Real-time Weather:** Fetches live weather data using the `wttr.in` API (No API key required).
* **Dynamic Clock:** High-precision digital clock for real-time tracking.
* **Persistent Task Manager:** A To-Do list that saves your data locally in the browser using `localStorage`.
* **Modern UI:** Premium Glassmorphism aesthetic with responsive CSS and Poppins typography.
* **Modular Architecture:** Clean separation of concerns (API, Storage, and UI logic) for easy maintenance.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables & Flexbox), JavaScript (ES6 Modules).
* **Design:** Glassmorphism UI with Backdrop-filter effects.
* **Version Control:** Git & GitHub (Feature Branch Workflow).
* **Deployment:** GitHub Pages.

## 🚀 Getting Started
1. Installation
Clone the repository to your local machine:

```bash git clone [https://github.com/xadeeb/ZenDash.git](https://github.com/xadeeb/ZenDash.git)```

2. Run the Project
Simply open index.html in any modern web browser or use the VS Code Live Server extension for the best development experience.
